Coding agents are no longer just about which model is strongest — the harness around the model matters too.

Artificial Analysis has introduced a new Coding Agent Index that benchmarks combinations of agent harnesses and models across three leading evaluations: SWE-bench Pro-Hard, Terminal-Bench v2, and SWE-Lancer. The first results make the model-vs-agent distinction very concrete.

A few notable signals:
- Cursor CLI with Opus 4.1 leads with a score of 61.
- Codex with GPT-5.5 and Claude Code with Opus 4.1 are close behind at 60.
- Cost per task varies widely: Cursor CLI with Composer 2 is shown around $0.10 per task with a score of 48, while Codex with GPT-5.5 is around $2.20 per task with a score of 60.

Why it matters: teams adopting AI coding tools need to evaluate the full system, not just the underlying model. Agent design, tool use, benchmark performance, token usage, and cost all shape the real developer experience.
