Anthropic Leaks: Claude Mythos & Claude Code Source Exposed

In late March 2026, Anthropic suffered two back-to-back leaks — a draft announcement of its next frontier model Claude Mythos and the full source code of Claude Code via npm.

Summary

In the span of five days at the end of March 2026, Anthropic was hit by two significant leaks that made headlines across the AI industry. First, a draft announcement for a next-generation model called Claude Mythos was accidentally published. Days later, the full source code of Claude Code — Anthropic's agentic coding tool — was exposed through an npm package. Together, these incidents raised questions about Anthropic's internal processes, even as the company is positioned as a safety-first AI lab.

The 3 Leaks at a Glance

1. Claude Mythos (March 26, 2026)

Anthropic accidentally published a draft announcement describing its next major model, Claude Mythos. The document described Mythos as more powerful than Opus and flagged elevated cybersecurity risks. The draft was quickly pulled, but not before it was widely cached and discussed.

2. Claude Code Source (March 31, 2026)

The source code of Claude Code was exposed via a published npm package containing roughly 512,000 lines of code across nearly 1,900 files. The package revealed the internal architecture of the tool — agent orchestration logic, automation pipelines, and internal component structure.

3. Two Leaks in Five Days

Multiple media outlets highlighted the unusual timing: Anthropic, a company that markets itself on safety and careful process, leaked both a frontier model announcement and its flagship developer tool's codebase within the same week. This put a spotlight on internal process failures.

What the Claude Code Leak Revealed

The Claude Code leak does not appear to have exposed customer data or credentials. However, it revealed a significant portion of the tool's internal architecture:

  • Agent components: internal agents and their orchestration logic
  • Automation pipelines: how Claude Code structures multi-step coding tasks
  • Internal tooling: supporting infrastructure that powers the product

The primary risk is strategic and competitive rather than a classical security breach. Competitors and researchers gained insight into how Anthropic structures its most visible developer-facing product.

Claude Mythos: The New Frontier Model

Based on the leaked draft, Claude Mythos represents a new tier above Opus:

  • Described as Anthropic's most capable model ever built
  • Significantly more advanced capabilities across reasoning, coding, and analysis
  • Flagged for higher cybersecurity risks — the safety evaluation noted elevated concerns
  • Anthropic reportedly confirmed the model exists and is being tested with a restricted group of early-access customers

Mythos appears to be Anthropic's answer to the frontier model race, positioning it against competitors like OpenAI's latest GPT family and Google's Gemini Ultra.

Key Takeaway

Mythos = new frontier model, Claude Code = leaked source code. Two incidents in five days that revealed both Anthropic's next strategic move and the internals of its developer platform, highlighting the gap between the company's safety-first messaging and its operational processes.
