Update 2026-04-13
Community report: Claude reasoning depth regression (Jan–Mar 2026)
A detailed user analysis documents a sharp drop in Claude's thinking depth between January and March 2026: median reasoning length fell from ~2 200 to ~600 chars, read/edit ratio dropped from 6.6× to 2.0×, and autonomous task completion degraded. The author calls for Anthropic to expose thinking-budget metrics via the API and offer a 'max thinking' tier for heavy engineering workflows.

Anthropic launches Claude Managed Agents (public beta)
A new suite of composable APIs to build and deploy cloud-hosted AI agents at scale. Features sandboxed execution, checkpointing, multi-agent orchestration, and scoped permissions. Deploy via Console, Claude Code, or CLI.

Update 2026-04-08
Meta launches Muse Spark — first model from Superintelligence Labs
Meta debuts Muse Spark, a proprietary multimodal reasoning model led by Alexandr Wang. Features thought compression (10× less compute than Llama 4 Maverick), multi-agent orchestration, and three reasoning modes. A major pivot away from Meta's open-source Llama strategy.
